# /writer:profile-edit - Edit Profile

@~/.claude/skills/shared-context.md

## Objective
Edit existing writer profile information.

## Process
1. Load current profile
2. Ask what to edit:
   - Brand information
   - Voice/tone
   - Products
   - CTAs
   - Case studies
3. Update selected section
4. Save changes to file
5. Reload profile in memory
6. Confirm changes saved

## Output
Profile updated successfully.
