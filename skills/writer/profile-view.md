# /writer:profile-view - View Profile

@~/.claude/skills/shared-context.md

## Objective
Display current writer profile information.

## Process
1. Check if profile loaded in memory
2. If not, list available profiles
3. Display profile details:
   - Brand information
   - Voice/tone settings
   - Products
   - CTAs
   - Case studies
4. Show profile file path

## Output
Complete profile information displayed.
